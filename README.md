🌱 Smart Farming Device Using IoT
📌 Overview

This project presents a Smart Farming Device using IoT that automates agricultural practices by monitoring environmental conditions and controlling irrigation systems in real-time. It reduces manual effort and improves farming efficiency through intelligent decision-making.

❗ Problem Statement

Traditional farming methods depend heavily on manual observation, which leads to:

Inefficient irrigation practices
Water wastage
Delay in detecting environmental changes
Increased labor work

This project provides a real-time monitoring and automated irrigation system to overcome these challenges and modernize farming practices.

🎯 Objectives
To automate irrigation based on soil moisture levels
To optimize water usage and reduce wastage
To display real-time sensor data on an interactive dashboard
To ensure healthy crop growth with precise irrigation management

⚙️ Key Features
📡 Real-time sensor monitoring with live alerts
💧 Automatic irrigation control using mobile app
⚡ Easy setup and low maintenance
🔋 Energy-efficient system
🚰 Pump activates only when needed and stops automatically
